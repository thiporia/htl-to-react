# htl-to-react

## 개요

`htl-to-react`는 HTL (HTML Template Language) 파일을 React 컴포넌트로 변환하는 도구입니다. 이를 통해 기존 HTL 템플릿을 React 기반 프로젝트에 더 쉽게 통합할 수 있습니다.
**현재 일부 태그에 대해서만 동작합니다.**

## 기능

- HTL 파일을 React 컴포넌트로 변환
- 사용자 정의 태그 및 속성 지원
